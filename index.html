Postman Integration Guide

Developers can test the OTP generation flow for partner registration using the following specs:

1. Request OTP Endpoint

Method: POST

URL: https://maasslabs.co.th/api/Auth/GetOTP

Body (JSON):

{ "EmailRequest": "partner@example.com" }

2. Response Handling

  • 200 OK: OTP sent to email; returns reference code.
  • 409 Conflict: Email already exists in tblHostModel.
  • 500 Internal Error: Failures in email service or DB persistence.
